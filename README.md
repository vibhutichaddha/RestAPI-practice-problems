# REST API & JSON Processing
This project demonstrates how to interact with the Github REST API using python and the 'requests' library. It covers making HTTP requests, processing JSON responses, handling errors, and saving API responses to JSON files.
## Learning Objective
- Understand the fundamentals of REST APIs
- Send HTTP requests using Python's 'requests' library
- Parse and process JSON requests
- Handle API errors and exceptions
- Build simple API-based Python Applications
## Topics Covered
- Introduction to REST APIs
- HTTP methods
- HTTP status codes
- Python requests library
- JSON processing
- Exception handling
- File handling
- Sorting Lists
-  Working with Dictionaries and Lists
## Tech Used
- Python 3
- Requests Library
- REST APIs
- JSON
